⚠️ 重要声明

本软件为「谢蔚安」角色专属智能体网站，所有功能、界面、提示词均围绕该角色深度定制开发。

不推荐将本软件用于其他人设角色的聊天。原因如下：
- 内置的人设、世界书、格式提示词、情侣空间、状态栏等均为谢蔚安量身设计
- 更换人设可能导致角色行为异常、格式错乱、功能失效
- 部分功能（如纪念日、反捧杀规则、危机应对等）与谢蔚安的设定深度绑定

如需使用其他角色，建议寻找通用型AI聊天前端，以获得更好的体验。

本声明由开发者心田发布

一、首次使用
1. 打开设置 → API 标签
2. 填入 API 地址（如 https://api.openai.com/v1）
3. 填入 API Key（sk-开头）
4. 选择模型（点"拉取"可查看可用模型）
5. 点"测试连接"确认成功后点"保存"
- 设置 → API标签页 → 角色自动发消息
- 开启后角色会按设定的时间间隔主动给你发消息
- 可设置间隔时间（1-180分钟）
- 角色会根据当前时间和场景自然地发起话题
- 无需手动点 ▼ 按钮
- 关闭后恢复手动模式

二、聊天基础
- 输入框打字 → 点发送（或回车）= 发消息
- 点左下角 ▼ 按钮 = 让角色回复
- 发完消息后点 ▼ 角色才会回复，不会自动回
- ▼ 按钮上的红色数字 = 待回复消息数
- 没发消息直接点 ▼ = 角色主动找你聊天

三、特殊消息
- 😊 按钮 = 发送表情包（需先在设置里添加）
- 🖼 按钮 = 发送图片
- 长按顶部头像 = 拍一拍
- 角色可发送文字图片 [图片：描述]

四、语音条
- 角色用"双引号"发送的内容会显示为语音条样式
- 点击语音条可展开查看文字内容（语音转文字）
- 再次点击收起

五、长按消息菜单
- 长按任意一条消息可弹出操作菜单
- 编辑：修改消息内容
- 重说：让角色重新生成这条回复（仅限角色消息）
- 删除：删除单条消息
- 手指移动超过一定距离会自动取消长按，不影响滚动
- 点击页面其他区域自动关闭菜单

六、状态栏
- 点击名称旁的"状态"按钮展开/收起
- 显示角色当前心情、动作、思绪、地点
- 每次角色回复时自动更新，不额外消耗API

七、朋友圈
- 点顶部 ◉ 进入朋友圈
- 上方输入框可以自己发朋友圈（支持配图）
- 右上角 📝 = 让角色发一条朋友圈
- 发布后角色和NPC会自动点赞+评论
- 在角色的朋友圈下评论，角色会回复你

八、情侣空间
- 点顶部 💕 进入情侣空间
- 顶部显示双方头像、名字、在一起天数
- 右上角 🖼 可上传情侣空间背景图
- 纪念日：可添加、编辑、删除自定义纪念日，显示倒计时
- 愿望清单：自己许愿或让角色许愿，发布后角色自动回复，可勾选完成
- 留言墙：自己留言或让角色留言，发布后角色自动回复
- 情绪曲线：记录双方情绪变化，自动生成折线图。你手动选择1-10分记录情绪后，角色会根据最近聊天内容自动分析并记录自己的情绪。也可以手动让角色记录。双方情绪数据以不同颜色的折线展示，直观看到情绪波动趋势
- 角色在聊天中能感知情侣空间的内容（包括情绪曲线）

九、查看TA的手机
- 在情侣空间左上角点 📱 进入
- 包含浏览记录、聊天记录、钱包、备忘录四个板块
- 点右上角"刷新"按钮获取最新内容
- 所有内容由AI根据角色设定和当前时间生成

十、知乎
- 发现 → 知乎：查看知乎首页和角色主页
- 角色主页可上传自定义背景图
- 角色头像与聊天界面自动同步
- 知乎 → 论坛标签页：自定义论坛
- 可设置论坛名称和规则/世界观
- 支持保存多个论坛预设，随时切换不同论坛
- 按"推荐/热榜/故事/问答"四个专栏分类浏览
- 点击刷新生成新帖子，可刷新评论区
- 帖子内容会根据论坛规则和世界观生成 - 刷新后的内容会自动保存，退出不会丢失

十一、商店与购物车
- 发现 → 商店：搜索商品，可发给角色付款或加入购物车
- ＋号菜单 → 购物车：查看已加入的商品
- ＋号菜单 → 自定义礼物：直接送礼物给角色
- 角色会根据设定决定是否付款

十二、后花园
- 发现 → 后花园：照料谢蔚安的药草花园
- 共6种植物：薄荷、迷迭香、薰衣草、洋甘菊、柠檬香蜂草、山茶花
- 每天可以给每棵植物浇水一次，增加成长值
- 也可以搞破坏（拔叶子），降低健康值
- 成长到100%会开花，健康降到0会枯萎，第二天自动重新生长
- 右上角 🏆 查看花园成就（浇水次数、破坏次数等）
- 角色在聊天中能感知花园动态

十三、Ta的房间
- 发现 → Ta的房间：偷偷翻看角色的私人物品
- 可以翻书桌、床、衣柜、保险箱四个位置
- 每天只能翻一次，翻完后角色会收到通知
- 角色会在聊天中含蓄地暗示他发现了痕迹
- 第二天可以再次翻找，内容每次不同

十四、群聊
- 发现 → 群聊：和角色、NPC一起群聊
- 右上角 ⚙ 进入群设置，可修改群名称、群世界观/背景
- 群成员：角色和你自动加入，可从NPC列表中添加其他成员
- 可为每个NPC成员单独设置头像
- 支持设置群聊背景图
- 在群里输入消息后点 ▼ 按钮，角色和NPC会根据上下文自然回复
- 不是所有人每次都会回复，AI会判断谁该参与讨论
- 群聊记录会随数据导入导出一起保存

十五、小说
- 发现 → 小说：生成你们的同人文
- 支持短篇小说和长篇连载两种类型
- 可输入主题关键词，也可留空随机生成
- 可选择字数（500/1000/2000/3000字）
- 支持自定义文风：右上角 🎨 添加文风描述和提示词
- 支持标签筛选：添加标签（如 #甜 #虐 #be）过滤内容
- 长篇连载可以续写下一章，自动承接上一章情节
- 右上角 ⭐ 查看收藏夹，可收藏喜欢的小说
- 支持复制小说内容
- 按"全部/短篇/长篇"分类浏览

十六、千纸鹤罐子
- ＋号菜单 → 千纸鹤：打开千纸鹤罐子
- 点「放一只进去」可以写一句话折成千纸鹤放进罐子
- 点「随机抽一只」会随机展开一只千纸鹤查看内容
- 罐子为空时抽取，角色会现场折一只新的给你

十七、视频通话
- ＋号菜单 → 视频通话：拨打视频通话
- 拨打后等待2-3秒自动接通（需要API已连接）
- 接通后可以在底部输入框打字和角色对话
- 角色会像真人接电话一样自然回应
- 左下角麦克风按钮可以静音
- 点红色挂断按钮结束通话
- 通话结束后，通话内容会自动写入聊天记录
- 通话时长会显示在界面上

十八、音乐播放器
- ＋号菜单 → 一起听歌：打开音乐播放器
- 支持上传mp3文件，可批量添加
- 支持列表循环、单曲循环、随机播放三种模式
- 支持上传lrc歌词文件，播放时自动滚动高亮
- 底部有迷你播放条，可快速控制播放
- 正在播放的歌曲信息和歌词会传递给角色
- 音乐数据存储在IndexedDB中，不占用localStorage空间
- 文件名格式建议：歌手 - 歌名.mp3（自动识别歌手和歌名）

十九、日记本
- 在情侣空间左上角点 📔 进入日记本
- 分为"我的日记"和"TA的日记"两个标签
- 我的日记：每天可以写一篇日记，写完后角色会自动在日记旁留下批注
- TA的日记：点"让TA写日记"，角色会根据最近聊天内容写一篇当天的日记
- 每人每天只能写一篇
- 可以回复角色的日记
- 日记本采用复古纸张风格设计
- 支持翻页浏览历史日记

二十、一起看书
- ＋号菜单 → 一起看书：打开书架
- 支持上传TXT格式小说文件（不超20MB）
- 可设置每页字数（200-5000字），默认800字
- 点"开始读"弹出浮窗，浮窗可拖动、可调整大小、可最小化
- 上一页/下一页翻页，自动记录阅读进度
- 开始看书时角色会收到通知，聊天中角色能感知当前阅读内容
- 可以边看书边聊天，角色会根据书中内容自然互动
- 退出看书时自动发送退出时间戳
- 书籍数据包含在JSON导出导入中
- 所有系统消息（时间戳）支持长按删除

二十一、启动页
- 打开页面时会显示启动画面
- 显示角色英文名「Everett」手写艺术字
- 加载完成后自动进入主界面

二十二、人设与世界书
- 已内置默认人设和世界书，开箱即用
- 可在"人设"和"世界书"标签里自由修改
- 点"恢复默认"可还原内置版本
- 不要清空，清空后角色会失去设定
- "备注"栏可写临时补充信息（如当前剧情状态）

二十三、格式提示词
- 控制角色回复的格式和风格
- 支持变量：{charName} {userName} {minReply}
- 一般不需要改，默认即可

二十四、记忆书
- 设置 → 记忆标签页
- 设置自动总结间隔（如每30条对话自动总结一次）
- 也可随时点"立即总结"手动触发
- 总结后的记忆会注入系统提示，帮角色记住重要事件
- 可编辑总结提示词控制总结风格
- 记忆条目可单独编辑（点 ✎）或删除
- 设为0则关闭自动总结

二十五、消息铃声
- 设置 → 外观 → 上传mp3文件
- 每次收到角色消息时播放提示音

二十六、常见问题
Q：角色不回复？
A：发完消息要点 ▼ 按钮触发回复。

Q：状态栏不更新？
A：去设置→格式→点"恢复默认"。

Q：角色照搬设定/重复对话？
A：已在格式提示词中内置防机械规则，如仍有问题可在备注栏写"不要复述人设，自然对话"。

Q：数据会丢失吗？
A：数据存在浏览器本地，建议定期导出备份。图片和音乐数据存储在IndexedDB中，更安全稳定。

💡 Temperature 建议 0.95-1.0。记忆条数建议 20-40。最少回复条数建议 2-4。
💡 备注栏可以写当前情境，随时修改不影响主人设。
💡 推荐模型：gpt-4o、claude-3.5-sonnet、deepseek-v3。性价比：gpt-4o-mini、deepseek-chat。
💡 情侣空间的内容不会被记忆总结收录，但角色聊天时能看到。
💡 情绪曲线建议每天记录1-2次，这样折线图更有参考价值。角色会根据聊天内容真实评估情绪，不会总是高分。
💡 后花园的植物每天会自动生长一点，即使不浇水也会慢慢长大，但浇水能加速很多。
💡 翻房间后角色的反应只会出现一次，不会反复追问，所以放心翻。
💡 音乐文件较大时建议用WiFi上传，音乐数据不会包含在JSON导出中。
💡 看书时角色不会每句话都提到书的内容，只在自然的时候才会聊到。翻页后角色能看到新的页面内容。
💡 书籍数据存储在localStorage中，大文件可能占用较多空间，建议单本不超过5MB。
